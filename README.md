## Instructions


If you use [nvm](https://github.com/nvm-sh/nvm) or [fnm](https://github.com/Schniz/fnm), execute:

```shell
nvm install
nvm use
```

Or:

```shell
fnm install
fnm use
```

To install and launch the project, run these commands:

```shell
yarn
yarn start
```


- To build and deploy, run the following commands:

```shell
yarn build
yarn deploy
```
